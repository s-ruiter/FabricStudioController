How to use:

  • Load Current: Load the current commands.json file
  • Save Changes: Save your changes (creates a backup automatically)
  • Validate JSON: Check if your JSON is valid before saving
  • Format JSON: Auto-format the JSON for better readability
  • Reset to Original: Discard changes and reload the original file

Important Notes:

  • Always validate JSON before saving to avoid breaking the application
  • A backup is automatically created when you save changes
  • Changes take effect immediately after saving
  • The editor provides syntax highlighting and error detection

Configuration Section:

  • _config.default_vm_filter: Default VM filter text
  • _config.default_vm_filter_placeholder: Placeholder text for filter input
