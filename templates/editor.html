How to use the editor:

  • Save Changes: Save your changes (creates a backup automatically)
  • Validate JSON: Check if your JSON is valid before saving
  • Format JSON: Auto-format the JSON for better readability

Important Notes:

  • Always validate JSON before saving to avoid breaking the application
  • A backup is automatically created when you save changes
  • Changes take effect immediately after saving
  • The editor provides syntax highlighting and error detection

üìã Commands.json Structure

Configuration Section (_config):

  • default_vm_filter: Default VM filter text used when no custom filter is entered (must contain "sru")

Command Properties:

  • command: REQUIRED - The actual SSH command to execute on the remote VM
  • responses: Optional - Dictionary of response patterns for interactive commands (format: {"pattern": "response"})
  • disconnect: Optional - Set to true if the command will disconnect/reboot the VM (default: false)
  • requires_extra_input: Optional - Set to true if the command needs additional user input (default: false)
  • prompt: Optional - The prompt text to show when requires_extra_input is true
  • warning: Optional - Set to true if the command should show 'Are you sure?' confirmation (default: false)

Usage Instructions:

  • Adding Commands: Add a new entry with a unique name as the key
  • Modifying Commands: Edit the properties in place
  • Restart Required: Restart the application after making changes to see them take effect
  • Config Changes: Changes to _config section take effect immediately without restart

Example Commands:

Simple Command:

{
  "System Status": {
    "command": "system status",
    "responses": {},
    "disconnect": false
  }
}

Interactive Command with User Input:

{
  "Change Password": {
    "command": "execute password admin {extra_input}",
    "requires_extra_input": true,
    "prompt": "Enter new admin password:",
    "responses": {"Password:": "{extra_input}"},
    "disconnect": false
  }
}

Dangerous Command with Confirmation:

{
  "Shutdown System": {
    "command": "system execute shutdown --no-interactive",
    "responses": {},
    "warning": true,
    "disconnect": true
  }
}
